# Portfólio Pessoal

Este projeto consiste no desenvolvimento de um **portfólio pessoal online**, criado com o objetivo de apresentar informações profissionais, projetos desenvolvidos e habilidades técnicas de forma organizada e acessível.

O portfólio funciona como uma vitrine profissional, permitindo que recrutadores e colaboradores conheçam melhor minhas experiências, tecnologias utilizadas e projetos realizados ao longo da minha trajetória de aprendizado na área de tecnologia.

O principal objetivo deste projeto é criar um espaço centralizado onde seja possível demonstrar minhas habilidades em desenvolvimento, além de reunir projetos e informações relevantes sobre minha formação e interesses na área de tecnologia.

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5** para estruturação das páginas
- **CSS3** para estilização e layout
- **JavaScript** para interatividade e funcionalidades dinâmicas

Entre as principais funcionalidades do projeto estão:
- Layout responsivo
- Navegação simples e intuitiva
- Apresentação de projetos desenvolvidos
- Estrutura organizada para facilitar a leitura das informações

## Importância do projeto

A criação deste portfólio representa um passo importante na construção da minha presença profissional na área de tecnologia. Além de demonstrar conhecimentos técnicos, o projeto também evidencia a capacidade de organizar informações e apresentar projetos de forma clara e estruturada.
